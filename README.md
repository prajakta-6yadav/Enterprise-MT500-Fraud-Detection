# Enterprise MT500 Fraud Detection System

## Overview
This project is an enterprise-grade SWIFT MT500 fraud detection platform designed to identify suspicious banking transactions using Machine Learning, Deep Learning, Rule-Based Detection, and Federated Learning techniques.

The system simulates real-world banking fraud analysis workflows used in financial institutions and cybersecurity environments.

---

## Features

- SWIFT MT500 transaction validation
- Fraud detection using ML models
- Deep Learning anomaly detection
- Rule Engine based fraud checks
- Federated Learning integration
- Real-time fraud alerts
- PDF investigation report generation
- Investigation dashboard
- SQLite database integration
